# Toán tử
cpp


 Trong C++, toán tử là các ký hiệu được sử dụng để thực hiện các phép tính số học hoặc logic trên các toán hạng. Toán tử cũng có thể được sử dụng để thực hiện các phép gán giá trị, nhập và xuất dữ liệu, so sánh các giá trị và nhiều hoạt động khác.

Các loại toán tử trong C++ được phân loại thành các nhóm sau:

### Toán tử Arithmetic (toán tử số học):
Các toán tử này được sử dụng để thực hiện các phép tính cơ bản như cộng, trừ, nhân, chia, chia lấy phần dư và các phép tính liên quan đến số học.

| Toán tử     | Miêu tả |
| ----------- | ----------- |
| +           | Cộng hai giá trị số học với nhau.     |
| -  | Cộng hai giá trị số học với nhau.        |
| * | Nhân hai giá trị số học với nhau. |
| / | Chia hai giá trị số học cho nhau. |
| % 	| Lấy phần dư khi chia hai giá trị số học cho nhau. |

### Toán tử Logical (toán tử logic):
Các toán tử này được sử dụng để thực hiện các phép tính logic và trả về một giá trị boolean true hoặc false.\

| Toán tử 	| Miêu tả |
| --------- | -------- |
| && 	| Phép AND logic - Trả về true nếu cả hai toán hạng là true. | 
| ! 	| Phép NOT logic - Đảo ngược giá trị của toán hạng từ true thành false và ngược lại. |
|   \|\|  | Phép OR logic - Trả về true nếu ít nhất một toán hạng là true. |


### Toán tử Assignment (toán tử gán giá trị):
Các toán tử này được sử dụng để gán giá trị của một biểu thức cho một biến.

| Toán tử | Miêu tả |
| ------ | ----- |
| = 	| Gán giá trị bên phải của biểu thức cho biến bên trái. | 
| += 	| Tăng giá trị bên phải của biểu thức vào giá trị của biến bên trái và lưu kết quả. |  
| -= 	| Giảm giá trị bên phải của biểu thức khỏi giá trị của biến bên trái và lưu kết quả. |
| *= 	| Nhân giá trị bên phải của biểu thức với giá trị của biến bên trái và lưu kết quả. |
| /= 	| Chia giá trị của biến bên trái cho giá trị bên phải của biểu thức và lưu kết quả. |
| %= 	| Lấy phần dư khi chia giá trị của biến bên trái cho giá trị bên phải của biểu thức và lưu kết quả. |

### Toán tử Comparison (toán tử so sánh):
Các toán tử này được sử dụng để so sánh hai giá trị và trả về một giá trị boolean true hoặc false.

| Toán tử |	Miêu tả |
| ---- | ---- |
| == 	| Trả về true nếu các toán hạng bằng nhau. |
| != 	| Trả về true nếu các toán hạng không bằng nhau. |
| < 	| Trả về true nếu toán hạng bên trái nhỏ hơn toán hạng bên phải. |
| > 	| Trả về true nếu toán hạng bên trái lớn |
