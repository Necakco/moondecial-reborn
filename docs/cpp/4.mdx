# Biến

C++

## Biến là gì
Trong lập trình, biến là một khái niệm quan trọng để lưu trữ giá trị hoặc kết quả tính toán. C++ là một ngôn ngữ lập trình hướng đối tượng và có kiểu dữ liệu cục bộ và toàn cục.
## I. Kiểu dữ liệu trong C++
 Trong C++, kiểu dữ liệu định nghĩa kiểu giá trị mà biến có thể lưu trữ. C++ cung cấp những kiểu dữ liệu căn bản và kiểu dữ liệu người dùng tự định nghĩa. Dưới đây là các kiểu dữ liệu trong C++:
### 1. Kiểu số nguyên (Integer)
Kiểu số nguyên được sử dụng để lưu trữ giá trị số nguyên. Trong C++, có nhiều loại kiểu số nguyên khác nhau, tùy thuộc vào kích thước và miền giá trị. Các kiểu dữ liệu nguyên khác nhau bao gồm:

- short: Kích thước của kiểu này được coi là 2 byte. Kiểu short có miền giá trị từ -32768 đến 32767.
- int: Đây là kiểu số nguyên phổ biến nhất trong C++, với kích thước 4 byte cho hầu hết các nền tảng. Miền giá trị của kiểu int thường là từ -2147483648 đến 2147483647.
- long: Đây là kiểu số nguyên có kích thước lớn hơn kiểu int, thường là 8 byte và miền giá trị từ -9223372036854775808 đến 9223372036854775807.
- long long: Đây là kiểu số nguyên có kích thước lớn nhất trong C++, có thể đạt đến 16 byte. Miền giá trị của kiểu này rất lớn, từ -9223372036854775808 đến 9223372036854775807.
### 2. Kiểu số thực (Float và Double)
C++ cũng cung cấp kiểu dữ liệu để đại diện cho các giá trị số thực. Có hai loại kiểu số thực chính trong C++: kiểu float và double.

- Kiểu float được sử dụng để lưu trữ các giá trị thập phân với độ chính xác khoảng 6 chữ số thập phân. Kích thước của kiểu float là 4 byte.
- Kiểu double cũng được sử dụng để lưu trữ các giá trị thập phân, nhưng với độ chính xác cao hơn và có kích thước 8 byte.
Ví dụ:
```cpp
float pi = 3.14159;
double e = 2.718281828459045;
```
## 3. Kiểu ký tự (Char)
Kiểu dữ liệu char chỉ có thể lưu trữ một ký tự duy nhất trong bảng mã ASCII. Kích thước của kiểu char là 1 byte và có thể giữ giá trị từ -128 đến 127 hoặc từ 0 đến 255.

Ví dụ:
```cpp
char grade = 'A';
cout << grade << endl;
```
## 4. Kiểu boolean
Kiểu boolean trong C++ đại diện cho các giá trị đúng/sai (true/false) hoặc 1/0. Kích thước của kiểu bool là 1 byte.

Ví dụ:
```cpp
bool isStudent = true;
if (isStudent) {
    cout << "This person is a student." << endl;
} else {
    cout << "This person is not a student." << endl;
}
```
Ngoài ra, trong C++ còn có các kiểu dữ liệu đặc biệt khác như void để không trả về giá trị, wchar_t để lưu trữ các ký tự Unicode, và các kiểu dữ liệu liên quan đến địa chỉ như ptrdiff_t, size_t và intptr_t. 
## II. Khai báo biến
### 1.1 Khai báo biến cục bộ
Khai báo biến cục bộ được sử dụng khi ta muốn sử dụng biến trong một phạm vi nhất định, ví dụ như giữa hai dấu ngoặc nhọn.

Cú pháp:
```
<kiểu dữ liệu> <tên biến>;
```
Ví dụ:
```cpp
int age;
float weight;
char gender;
```
### 1.2 Khai báo biến toàn cục
Khai báo biến toàn cục được sử dụng để lưu trữ giá trị trong suốt chương trình. Biến toàn cục được khai báo bên ngoài tất cả các hàm và có thể được sử dụng bởi bất kỳ hàm nào trong chương trình.

Cú pháp:
```
<kiểu dữ liệu> <tên biến>;
```
Ví dụ:
```cpp
int total_marks = 100;
float PI = 3.14;
char first_name[] = "John";
```
### 2. Gán giá trị cho biến
Khi khai báo một biến, nó không có giá trị ban đầu. Tuy nhiên, ta có thể gán một giá trị cho biến bằng toán tử gán ( = ).

Ví dụ:
```cpp
int n; 
n = 10; //gán giá trị 10 cho biến n
```
### 3. Hiển thị giá trị của biến
Để hiển thị giá trị của một biến, ta sử dụng lệnh cout.

Ví dụ:
```cpp
int n = 10;
cout << n << endl; //hiển thị giá trị của biến n ra màn hình
```
