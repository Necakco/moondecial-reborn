# Chương Trình Đầu Tiên

c++ 

## 1. Chương trình đầu tiên , làm quên với file CPP
Đây là hướng dẫn đơn giản về cách viết chương trình đầu tiên sử dụng ngôn ngữ lập trình C++.
### Bước 1: Tạo một tệp tin mới
Mở trình biên tập Code và tạo một tệp tin mới. Theo quy ước thì tệp tin có đuôi là .cpp
### Bước 2: Viết code
Sau khi tạo được file, ta sẽ truyền cho máy tính những yêu cầu, thông qua những lệnh lập trình ngôn ngữ C++.
Ví dụ ở đây chúng ta sẽ in ra màn hình console hai từ "Hello World". Để làm được điều này, ta sẽ viết mã lệnh sau:
```cpp
#include <iostream> // Thư viện chuẩn để in và nhận input từ console trong C++

int main() { // khai báo hàm main (hàm chính)
    std::cout << "Hello World" << std::endl; // In ra dòng chữ "Hello World"
    return 0; // Trả về giá trị 0 kết thúc chương trình
}
```
### Bước 3: Biên dịch chương trình
Sau khi hoàn thành viết code, ta tiến hành biên dịch chương trình. Quá trình biên dịch sẽ chuyển các lệnh lập trình sang ngôn ngữ máy tính có thể hiểu được.
Bạn có thể sử dụng các trình biên dịch như GCC, Clang ... nếu bạn sài các ide khác thì hãy tham khảo cách setup chạy .cpp file tại Đây
### Bước 4: Chạy chương trình
Nếu không có lỗi trong quá trình biên dịch, bạn có thể chạy chương trình và kiểm tra kết quả.
Và đó là bài hướng dẫn cơ bản để viết chương trình Hello World bằng C++.
## 2. Một số câu lệnh cơ bản của C++
C++ cung cấp nhiều câu lệnh để thực hiện các tác vụ khác nhau. Dưới đây là một số câu lệnh cơ bản trong C++:

Câu lệnh xuất dữ liệu ra màn hình(console)
```cpp
std::cout << "Hello world!\n";
```
\n là xuống dòng trong console
Câu lệnh nhập dữ liệu từ bàn phím cin: dùng để nhận dữ liệu nhập từ bàn phím vào biến.
```cpp
int value;
std::cin >> value;
```
VD :
```cpp
#include <iostream>

int main() {
    int age;
    std::cout << "Nhap tuoi: ";
    std::cin >> age;  // Nhập giá trị tuổi từ bàn phím
    std::cout << "Tuoi cua ban la: " << age << std::endl;
    return 0;
}
```
cerr: dùng để in ra thông báo lỗi ra console.
```cpp
#include <iostream>
#include <cstdlib>

int main() {
    int num = 0;
    if (num == 0) {
        std::cerr << "Loi: Khong the chia mot so cho 0.\n";
        exit(1);
    }
    int result = 10 / num;
    std::cout << "Ket qua: " << result << std::endl;
    return 0;
}
```
vì câu lệnh này có liên quan đến IF | ELSE nên bài sau các bạn vẫn nên xem lại

getchar: dùng để đợi đến khi người dùng nhấn phím Enter để tiếp tục thực thi chương trình.
```cpp
#include <cstdio>  // hoặc #include <stdio.h>

int main() {
    std::cout << "Chuong trinh da thuc hien xong!\n";
    getchar();  // Chờ đợi nhập kí tự và nhấn Enter để thoát chương trình.
    return 0;
}
```
Lưu ý, khi sử dụng chỉ số cout, cin, cerr, có thể sẽ gặp điều kiện std::trước tên, do đó cần phải khai báo namespace using namespace std;ở đầu chương trình. 

## [Thực hành] Xây dựng chương trình đơn giản

Để xây dựng một chương trình đơn giản bằng C++, bạn có thể làm theo các bước sau:

### Bước 1:
Mở trình biên dịch (Visual Studio Code, Dev-C++, c++,...) tùy vào sở thích của bạn.
### Bước 2:
Tạo một file mới với phần mở rộng là ".cpp".
### Bước 3:
Sau đó, nhập mã nguồn vào trong file đó. Dưới đây là một ví dụ cho Chương trình in ra chuỗi Hello World!:
```cpp
#include <iostream>
using namespace std;

int main() {
   cout << "Hello World!";
   return 0;
}
```
### Bước 4:
Lưu và biên dịch code. Biên dịch sẽ tạo ra một tệp tin ngoài file.cpp đó là file.exe.
### Bước 5:
Chạy chương trình.
Chúc mừng bạn đã tạo thành công một chương trình đơn giản bằng C+
