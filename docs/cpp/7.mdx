# Hàm

c++

Hàm (Function) trong C++ là một khối mã có thể được tái sử dụng bằng cách gọi nó từ các phần khác của chương trình. Nó sẽ chứa một tập hợp lệnh để thực hiện một tác vụ hoặc tính toán giá trị và trả về kết quả.
## 1. Khai báo hàm
Khai báo hàm trong C++ có ba đặc điểm sau: Tên hàm – Một tên cho hàm của bạn, một cái tên đã được chọn sao cho nó cung cấp ý nghĩa cho những gì hàm của bạn được thiết kế để làm. Kiểu trả về – Loại giá trị trả về khi hàm được gọi. Tham số – Các đối tượng chúng ta muốn truyền vào hàm. Cú pháp
```cpp
[kiểu_trả_về] [tên_hàm](tham_số){
  // Các câu lệnh
}
```
Trong đó:

- kiểu_trả_về là kiểu dữ liệu được trả về bởi hàm.
- tên_hàm là tên của hàm.
- tham_số là danh sách các đối số mà hàm sẽ sử dụng.

Ví dụ ta khai báo hàm tính tổng 2 số nguyên:
```
int sum(int num1, int num2){
    int result = num1 + num2;
    return result;
}
```
Hàm này có tên là sum.

Hàm này trả về một giá trị kiểu integer (int).

Hàm này có hai tham số được truyền vào và kiểu của chúng đều là integer.

Ngoài ra, có thể khai báo hàm mà không có giá trị trả về (kiểu void), ví dụ:
```cpp
void printMessage(string message) {
  cout << message << endl;
}
```

## 2. Gọi hàm

Để gọi hàm trong chương trình C++, chỉ cần sử dụng tên hàm và truyền đối số (nếu có).
```cpp
int main() {
    int a = 5, b = 10;
    int c = sum(a, b);
    cout << "Tong hai so la: " << c << endl;

    return 0;
}
```
## Hàm đệ quy
Hàm đệ quy là hàm gọi lại chính nó, thực hiện đệ quy để giải quyết một vấn đề. Hàm đệ quy phải có điều kiện dừng để tránh xảy ra vòng lặp vô hạn.

Ví dụ tính giai thừa bằng đệ quy:
```cpp
int factorial(int n) {
    if (n == 0) {
        return 1;
    } else {
        return n * factorial(n - 1);
    }
}
int main() {
    int n = 7;
    int result = factorial(n);
    cout << "Giai thua cua " << n << " la: " << result << endl;

    return 0;
}
```
Cuối cùng, hàm là một khối mã có thể được tái sử dụng bằng cách gọi nó từ phần khác của chương trình (ví dụ trong hàm main).
