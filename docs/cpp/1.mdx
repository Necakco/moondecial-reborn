## 3. Cài đặt trình biên dịch: GCC
GCC (GNU Compiler Collection) là một trình biên dịch mã nguồn mở rộng và hỗ trợ cho nhiều ngôn ngữ lập trình, bao gồm C++. Để cài đặt GCC, bạn có thể thực hiện các bước sau:

### Trên hệ điều hành Linux
Mở terminal và nhập lệnh sau để cài đặt GCC:
```bash
sudo apt-get update
sudo apt-get install build-essential
```
### Trên hệ điều hành Windows
Tải xuống phiên bản MinGW của GCC tại đây.
